Оборудование в прокате

Не нужно дублировать блоки для 2-го, 3-го экземпляра — в шаблон Word вставьте одну переменную:

  • {{EQUIPMENT_LINES}} — каждая позиция с новой строки
  • {{EQUIPMENT_INLINE_PLUS}} — всё в одну строку
  • {{EQUIPMENT_NUMBERED}} — нумерованный список
  • {{EQUIPMENT_LINES_SHORT}} — коротко (название и S/N)

Формат текста по оборудованию уже зашит в CRM: в {{EQUIPMENT_LINES}} подставляется полный блок (оценка, состояние, все тарифы суток и месяцев, точка). Для таблицы по слотам остаются переменные {{EQUIPMENT_1_TITLE}} … {{EQUIPMENT_50_…}}.

Вставляйте в Word именно в двойных фигурных скобках. При скачивании документа по прокату подставятся данные клиента и проката.

Ниже перечислены переменные для каждой из до 50 позиций оборудования в прокате ({{EQUIPMENT_1_TITLE}} … {{EQUIPMENT_50_PRICE_SUBSEQUENT_MONTHS}}). Для короткого варианта используйте {{EQUIPMENT_LINES}}.

Как это работает вместе

  1. Шаблоны — выше: загружаете .docx из Word. Переменные — в блоке «Все переменные для шаблонов» (все примеры с описанием).
  2. Журнал прокатов — у проката нажимаете иконку документа, выбираете шаблон.
  3. Скачать документ (.docx) — в модалке скачиваете заполненный документ. Всё вместе в конце цепочки.
